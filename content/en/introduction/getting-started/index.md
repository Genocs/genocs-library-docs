## Problem Statement

<p>Ever gone through the pain of setting up a new project from scratch each time?</p>
<p><b>Genocs Library</b> aims to
address this pain point by offering starting points for various technologies that can help save
hours of development and research time for developers and teams.</p>

## Solution

<p><b>Genocs Library</b> offers a library framework, CLI, and templates with the latest packages and services your projects need. We provide clean, well-structured templates following standard coding practices and Clean Architecture principles that make your development experience seamless.
</p>
<p class="code-green">Most importantly, <b>Genocs Library</b> is completely FREE to use!</p>

As the name suggests, the idea is to provide a full-stack, end-to-end solution to kick-start products of any scale following a modularized monolith architecture with clear design, readable code, and scalability out of the box. Bottom line: with <b>Genocs Library</b>, your new projects can be up and running quickly.

## Vision

<b>Genocs Library</b> is not a single repository or project; it is a collection of libraries, templates, and a CLI—you choose what fits. In the long term, <b>Genocs Library</b> aims to offer multiple API projects in technologies such as .NET, Node.js, and Go, and client applications on stacks like Blazor, Vue.js, React, and Angular.

<b>Genocs Library</b> aims to be a one-stop resource for developers across technologies. That vision depends on community adoption and contributions.

For the initial releases, <b>Genocs Library</b> includes an API project on <b>.NET 10</b> and a Blazor WebAssembly front end that consumes that Web API.
