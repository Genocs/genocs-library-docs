# Home

The Home module is a part of the Genocs Library's Blazor WebAssembly Template. It provides a set of pages to manage the home page in the application. The module includes the following pages:


- **Home**: The main page of the application.

For the home page, you can view and edit the following information:

- **Title**: The title of the home page.
- **Description**: A description of the home page.
- **Content**: The content of the home page.
